# Lab 2
**Repo for Web Systems lab 2**

This lab consists of a story within the Pokémon world.

You are a young person excited to visit the professor and get a new companion.

Depending on the option you choose in each section of the story, the ending you get may *change*.

## Instructions:
1. Open the index.html file in your preferred browser.

- To open it, you can drag the file from the file explorer into the browser.

2. You're all set! Once the file is open, you can now explore the story.
